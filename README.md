# Horseclub

Система онлайн‑записи и расписания для конных клубов (Express + PostgreSQL + Socket.IO + vanilla JS).

## Быстрый старт (локально)

### 1) Установи зависимости

```bash
npm install
```

### 2) Создай `.env`

Скопируй пример и отредактируй:

```bash
copy .env.example .env
```

В `.env` поменяй минимум:
- `JWT_SECRET` (любой длинный секрет)
- `DATABASE_URL` (или `PGHOST/PGUSER/...`)

### 3) Подними PostgreSQL и создай базу

Тебе нужна база с именем `horseclub`.

### 4) Создай таблицы

```bash
npm run db:schema
```

### 4.1) (Опционально) Заполни тестовыми данными

Создаст пользователей и немного примеров, чтобы сразу всё “пощупать”.

Логины:
- `admin` / `admin123`
- `user` / `user123`

```bash
npm run db:seed
```

### 5) Запусти сервер

```bash
npm start
```

Открой `http://localhost:3000`.

## Важные принципы проекта

- Секреты (JWT, доступ к БД) **не храним в коде** — только в `.env`.
- Схема БД хранится в `sql/schema.sql`, чтобы проект поднимался на чистой машине.
- После первого запуска (когда создан админ) публичная регистрация закрывается. Для разработки можно временно включить `ALLOW_PUBLIC_REGISTER=true` в `.env`.
